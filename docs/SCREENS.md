# Экраны — Мафия Онлайн

Дизайн-документ для всех экранов приложения. Общая тема: тёмная, атмосферная (ночной город, мафия, детектив). Мобильный дизайн в приоритете (игроки используют телефоны), но должно работать и на десктопе.

## 1. Главная страница (HomeScreen)

**Назначение**: Точка входа. Создать или присоединиться к комнате.

**Элементы**:
- Логотип / название "МАФИЯ" (крупно, стилизовано)
- Подзаголовок "Онлайн" (мелко)
- Поле ввода: "Ваше имя" (обязательное)
- Кнопка: "Создать комнату" (основная, крупная)
- Разделитель "или"
- Поле ввода: "Код комнаты" (6 символов, uppercase)
- Кнопка: "Присоединиться" (вторичная)
- Атмосферный фон (силуэт города ночью, или тень в шляпе)

**Состояния**:
- Если в URL есть `?room=XXXXXX` — код комнаты предзаполнен, фокус на поле имени
- Валидация: имя 2–20 символов, код 6 символов
- Ошибка подключения: "Комната не найдена" / "Комната уже в игре"

## 2. Создание комнаты (CreateRoomScreen)

**Назначение**: Хост настраивает параметры игры перед созданием комнаты.

**Элементы**:

### Секция "Роли"
Для каждой роли — карточка со:
- Иконка роли
- Название
- Краткое описание (1 строка)
- Тогл вкл/выкл (для булевых ролей: Дон, Комиссар, Врач, Маньяк, Адвокат)
- Счётчик +/- (для Мафии — количество)

Порядок ролей на экране:
1. Мафия (счётчик, мин. 1)
2. Дон мафии (тогл)
3. Адвокат (тогл)
4. Комиссар (тогл)
5. Врач (тогл)
6. Маньяк (тогл)

Информационная строка внизу секции: "Для X ролей нужно минимум Y игроков. Остальные будут мирными жителями."

### Секция "Голосование"
- Переключатель: Открытое / Закрытое
- Подсказка: "Открытое — все видят голоса. Закрытое — результат после завершения."

### Секция "Таймеры"
- Мастер-тогл: Таймеры вкл/выкл
- Если включены:
  - Ночное действие: слайдер/поле 15–120 сек (по умолчанию 30)
  - Обсуждение: слайдер/поле 30–600 сек (по умолчанию 120)
  - Голосование: слайдер/поле 15–120 сек (по умолчанию 60)
  - Прощальное слово: слайдер/поле 10–120 сек (по умолчанию 30)

### Секция "Дополнительно"
- Тогл: "Раскрывать роль убитого" (по умолчанию: да)
- Тогл: "Озвучка" (по умолчанию: да, только для хоста)

### Кнопка внизу
- "Создать комнату" (крупная, фиксирована внизу)

## 3. Лобби (LobbyScreen)

**Назначение**: Зал ожидания. Игроки подключаются, хост запускает игру.

**Элементы**:

### Верхний блок
- Код комнаты (крупно, моноширинный шрифт, например `A 7 K X 3 M`)
- Кнопка "Копировать код" (иконка копирования)
- QR-код (средний размер, можно развернуть тапом)

### Список игроков
- Каждый игрок: аватар-цвет (круг с инициалом) + имя
- У хоста — бейдж "Хост" или иконка короны
- Анимация появления нового игрока
- Индикатор подключения (зелёная/жёлтая/красная точка)
- Хост может кикнуть игрока (иконка крестика, с подтверждением)

### Информационная строка
- "Игроков: 5 | Нужно минимум: 6" (если недостаточно для выбранных ролей)
- Или "Игроков: 8 | Можно начинать!" (зелёная)

### Настройки (только хост)
- Кнопка "Настройки" → открывает экран CreateRoomScreen для редактирования
- Изменения применяются сразу и видны всем игрокам в лобби

### Кнопки внизу
- Хост: "Начать игру" (активна, если достаточно игроков) + "Выйти"
- Игрок: "Выйти из комнаты"

**Состояния**:
- Кнопка "Начать" неактивна (серая) + подсказка при наведении, если мало игроков
- При изменении настроек хостом — все видят уведомление "Хост изменил настройки"

## 4. Раздача ролей (RoleRevealScreen)

**Назначение**: Каждый игрок узнаёт свою роль.

**Элементы**:
- Карточка роли по центру экрана:
  - Анимация переворота (рубашка → лицо)
  - Иконка роли (крупная)
  - Название роли
  - Цвет карточки зависит от роли:
    - Мафия/Дон/Адвокат → красная рамка/свечение
    - Комиссар → синяя
    - Врач → зелёная
    - Маньяк → фиолетовая
    - Мирный → серая/белая
  - Описание способности (2–3 строки)
- Для мафии: дополнительный блок "Ваша команда:" + список имён мафиози
- Кнопка "Готов" (внизу)
- Индикатор: "Ожидание остальных... 5/8 готовы"

**Важно**: На экране мафиози видят имена всей команды. Остальные видят только свою роль.

## 5. Ночь — Ожидание (NightWaitScreen)

**Назначение**: Экран для игроков, чья роль сейчас не активна.

**Элементы**:
- Тёмный фон с анимацией (звёзды, луна, силуэт города)
- Надпись: "Город спит..." (крупно, по центру)
- Подпись: "Ночь X" (номер ночи)
- Информация: кто сейчас делает выбор (без раскрытия имён):
  - "Мафия делает выбор..."
  - "Дон проводит расследование..."
  - "Адвокат готовит защиту..."
  - "Маньяк выходит на охоту..."
  - "Комиссар ведёт расследование..."
  - "Врач спасает жизни..."
- Пульсирующая анимация или индикатор ожидания

## 6. Ночь — Мафия (NightMafiaScreen)

**Назначение**: Мафия и Дон выбирают жертву.

**Видят**: Только игроки с ролями Мафия, Дон (Адвокат видит, но не голосует за жертву — у него свой ход).

**Элементы**:
- Заголовок: "Мафия просыпается" + иконка
- Подзаголовок: "Выберите жертву"
- Таймер (если включён)
- Список живых игроков (кроме мафии):
  - Имя + аватар-цвет
  - При тапе — выбор (подсвечивается красным)
  - Если другой мафиози уже проголосовал за этого игрока — показать иконку/метку
- Блок "Ваша команда":
  - Список мафиози с аватарами
  - Рядом с каждым — за кого он голосует
- Кнопка "Подтвердить" (активна после выбора)
- Если все мафиози выбрали одного — выбор автоматически подтверждается
- При ничьей — Дон имеет решающий голос (показать подсказку)

## 7. Ночь — Дон (NightDonScreen)

**Назначение**: Дон проверяет, является ли игрок Комиссаром.

**Видит**: Только Дон.

**Элементы**:
- Заголовок: "Дон проводит расследование"
- Подзаголовок: "Кто из них Комиссар?"
- Таймер
- Список живых игроков (кроме мафии)
- При выборе → показать результат:
  - "Комиссар!" (зелёная галочка / иконка значка)
  - "Не комиссар" (красный крестик)
- Кнопка "Понятно" после показа результата

## 8. Ночь — Адвокат (NightAdvocateScreen)

**Назначение**: Адвокат выбирает мафиози для защиты от проверки Комиссара.

**Видит**: Только Адвокат.

**Элементы**:
- Заголовок: "Адвокат готовит защиту"
- Подзаголовок: "Кого защитить от проверки?"
- Таймер
- Список живых мафиози (имена, аватары):
  - Включая самого себя
  - Включая Дона (если жив)
  - Включая обычных мафиози
- Выбор одного → подтверждение
- Пояснение: "Если Комиссар проверит защищённого — увидит 'Мирный'"
- Можно пропустить (кнопка "Не защищать никого")

## 10. Ночь — Комиссар (NightCommissionerScreen)

**Назначение**: Комиссар проверяет игрока.

**Видит**: Только Комиссар.

**Элементы**:
- Заголовок: "Комиссар ведёт расследование"
- Подзаголовок: "Кого проверить?"
- Таймер
- Список живых игроков (кроме себя)
- При выборе → показать результат:

  - "Мирный" (зелёный, иконка голубя) — если мирный, маньяк, дон или защищённый адвокатом

- Комиссар может запомнить предыдущие проверки (лог проверок на экране?)

## 11. Ночь — Врач (NightDoctorScreen)

**Назначение**: Врач выбирает, кого лечить.

**Видит**: Только Врач.

**Элементы**:
- Заголовок: "Врач спасает жизни"
- Подзаголовок: "Кого вылечить этой ночью?"
- Таймер
- Список живых игроков (включая себя):

- Зелёное свечение/акцент

## 12. День — Результаты ночи (DayResultsScreen)

**Назначение**: Показать, что произошло ночью.

**Элементы**:
- Фон: переход от тёмного к светлому (рассвет)
- Заголовок: "Наступает день" + "День X"
- Блок результатов:
  - **Если кто-то убит**:
    - Карточка убитого: аватар + имя + (роль, если настройка включена)
    - Если убиты двое (мафия + маньяк): две карточки
    - Анимация "затемнения" карточки
    - Подпись: "Этой ночью был убит {имя}" / "Этой ночью были убиты {имя1} и {имя2}"
  - **Если никто не убит**:
    - Сообщение: "Этой ночью в городе было тихо. Никто не погиб."
    - Иконка: щит / солнце
- Кнопка хоста: "Начать обсуждение"
- Игроки ждут пока хост нажмёт кнопку

## 13. День — Обсуждение (DiscussionScreen)

**Назначение**: Фаза обсуждения перед голосованием.

**Элементы**:
- Заголовок: "Обсуждение" + "День X"
- Таймер обсуждения (если включён, обратный отсчёт)
- Список живых игроков (круглые аватары в ряд или сеткой):
  - Зелёный = жив
  - Серый/затемнённый = мёртв
- Список мёртвых (мелко, внизу или свёрнутый)
- Кнопка хоста: "Начать голосование" (можно нажать до конца таймера)
- Подсказка: "Обсуждайте, кто может быть мафией"

---

## 14. День — Голосование (VotingScreen)

**Назначение**: Игроки голосуют за изгнание.

**Элементы**:
- Заголовок: "Голосование" + "День X"
- Таймер (если включён)
- Список живых игроков (кандидаты для изгнания):
  - Аватар + имя
  - Кнопка голоса (при тапе — выбор)
  - **Открытое голосование**: Рядом с кандидатом — список/счётчик кто за него проголосовал
  - **Закрытое голосование**: Только "Вы проголосовали за {имя}" без чужих голосов
- Кнопка: "Не голосовать" / "Пропустить" (можно воздержаться)
- Можно изменить голос до конца таймера (или пока не подтвердил)
- Индикатор: "Проголосовали: 5/8"

### Результат голосования (VoteResultOverlay)
- Показать результаты:
  - Список кандидатов + количество голосов
  - **Открытое**: расшифровка — кто за кого
  - **Закрытое**: только числа
- Если есть лидер — "Город изгоняет {имя}"
- Если ничья — "Город не пришёл к единому мнению. Никто не изгнан."
- Кнопка хоста: "Продолжить" → переход к прощальному слову или следующей ночи

---

## 15. День — Прощальное слово (LastWordScreen)

**Назначение**: Изгнанный игрок произносит последние слова.

**Элементы**:
- Аватар и имя изгнанного (крупно, по центру)
- Роль (если настройка "раскрывать роль" включена)
- Надпись: "Прощальное слово"
- Таймер (если включён)
- Все остальные видят этот экран и ждут
- Кнопка хоста: "Продолжить" (завершить фазу прощального слова)

---

## 16. Экран мёртвого игрока (DeadScreen)

**Назначение**: Экран для выбывших игроков.

**Элементы**:
- Заголовок: "Вы выбыли из игры"
- Ваша роль (карточка)
- Текущая фаза игры (ночь/день, номер)
- Текстовый лог событий:
  - "Ночь 2: кто-то был убит"
  - "День 2: {имя} изгнан голосованием"
  - Но НЕ видит роли живых и секретные действия
- Список живых и мёртвых игроков (без ролей живых)
- Атмосфера: полупрозрачный/затемнённый интерфейс, иконка привидения
- Анимация: лёгкая "призрачность" (opacity, размытие)

---

## 17. Конец игры (GameOverScreen)

**Назначение**: Итоги игры. Раскрытие всех ролей.

**Элементы**:
- Заголовок: "Игра окончена!"
- Блок победителя:
  - "Мафия победила!" (красный, иконка мафии) или
  - "Мирные жители победили!" (синий, иконка щита) или
  - "Маньяк победил!" (фиолетовый, иконка ножа)
- Анимация победы (конфетти? пульсация?)
- Таблица всех игроков:
  - Аватар + имя
  - Роль (раскрыта)
  - Статус: жив / убит ночью N / изгнан днём N
  - Цвет строки = команда (красный для мафии, синий для мирных, фиолетовый для маньяка)
- Кнопки:
  - Хост: "Новая игра" (возврат в лобби с теми же игроками)
  - Все: "Выйти"

## 18. Модальные окна и оверлеи

### Подтверждение действия
- "Вы уверены, что хотите [убить/проверить/вылечить] {имя}?"
- Кнопки: "Да" / "Отмена"

### Подключение потеряно (ConnectionLostOverlay)
- "Соединение потеряно"
- "Попытка переподключения... (15 сек)"
- Спиннер
- Кнопка: "Выйти"

### Игрок отключился (для хоста) (PlayerDisconnectedModal)
- "{имя} отключился"
- "Ожидание переподключения: 30 сек"
- Кнопки: "Подождать" / "Кикнуть"

### Кик игрока (KickConfirmModal)
- "Исключить {имя} из комнаты?"
- Кнопки: "Исключить" / "Отмена"

### QR-код развёрнутый (QRCodeModal)
- QR-код на весь экран (удобно для сканирования)
- Код комнаты текстом под QR
- Кнопка закрытия

### Настройки комнаты (RoomSettingsModal)
- Те же поля что в CreateRoomScreen
- Доступно только хосту
- Кнопка "Сохранить"

| Устройство | Раскладка |
|-----------|-----------|

## Анимации

| Элемент | Анимация |
|---------|----------|
| Переход день/ночь | Плавная смена фона (светлый ↔ тёмный), 0.5–1 сек |
| Карточка роли | Flip-анимация (переворот 3D) |
| Появление игрока | Fade-in + slide-up |
| Убийство | Карточка "разбивается" или тускнеет |
| Голосование | Иконки "летят" к кандидату (открытое) |
| Таймер | Пульсация при < 5 сек |
| Победа | Конфетти + пульсация заголовка |
