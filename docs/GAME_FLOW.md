# Игровые сценарии — Мафия Онлайн

Пошаговые сценарии всех процессов в игре.

---

## Сценарий 1: Создание комнаты и подключение

### Действия хоста
```
1. Открывает приложение → HomeScreen
2. Вводит имя → "Артём"
3. Нажимает "Создать комнату" → CreateRoomScreen
4. Настраивает роли:
   - Мафия: 2
   - Дон: вкл
   - Комиссар: вкл
   - Врач: вкл
   - Маньяк: выкл
   - Адвокат: выкл
5. Настраивает таймеры:
   - Ночное действие: 30 сек
   - Обсуждение: 120 сек
   - Голосование: 60 сек
   - Прощальное слово: 30 сек
6. Голосование: Открытое
7. Раскрывать роль убитого: Да
8. Нажимает "Создать комнату"
9. → LobbyScreen
   - Код: A7KX3M
   - QR-код
   - В списке: "Артём (Хост)"
```

### Действия игрока
```
1. Открывает приложение → HomeScreen
   - Вариант А: сканирует QR → автозаполнен код
   - Вариант Б: вводит код "A7KX3M"
2. Вводит имя → "Марина"
3. Нажимает "Присоединиться"
4. → LobbyScreen
   - Видит список игроков: Артём (Хост), Марина, ...
   - Ждёт начала игры
```

### Что видит хост при подключении игрока
```
- Звуковой/вибро-сигнал (необязательно)
- Новый игрок появляется в списке с анимацией
- Счётчик обновляется: "Игроков: 2/6 (нужно минимум 6)"
```

## Сценарий 2: Запуск игры

### Предусловия
- В лобби 8 игроков
- Роли: 2 Мафии, 1 Дон, 1 Комиссар, 1 Врач, 3 Мирных

### Шаги
```
1. Хост видит: "Игроков: 8 | Можно начинать!"
2. Нажимает "Начать игру"
3. Система валидирует: ролей (2+1+1+1=5 спец.) ≤ 8 игроков. ОК.
4. Система распределяет роли случайным образом:
   - Артём → Мирный
   - Марина → Мафия
   - Денис → Комиссар
   - Алина → Мирный
   - Костя → Дон
   - Юля → Врач
   - Максим → Мафия
   - Катя → Мирный
5. Каждый игрок → RoleRevealScreen со своей ролью
6. Мафиози (Марина, Костя, Максим) дополнительно видят:
   "Ваша команда: Марина (Мафия), Костя (Дон), Максим (Мафия)"
7. Каждый нажимает "Готов"
8. Когда все готовы → Первая ночь
```

## Сценарий 3: Ночная фаза (полная)

### Пример: Ночь 1

**Устройство хоста** (TTS озвучка):

```
Шаг 1: Озвучка "Город засыпает. Наступает ночь."
        Все игроки → NightWaitScreen "Город спит..."

Шаг 2: Озвучка "Мафия просыпается."
        Марина, Костя, Максим → NightMafiaScreen
        Остальные → NightWaitScreen "Мафия делает выбор..."
        
        Марина голосует за Дениса
        Костя голосует за Артёма
        Максим голосует за Дениса
        → Большинство: Денис (2 голоса)
        → Мафия подтверждает: жертва = Денис
        
        Озвучка "Мафия засыпает."

Шаг 3: Озвучка "Дон просыпается."
        Костя → NightDonScreen
        Остальные → NightWaitScreen "Дон проводит расследование..."
        
        Костя проверяет Юлю → "Не комиссар"
        Костя нажимает "Понятно"
        
        Озвучка "Дон засыпает."

Шаг 4: (Адвокат пропускается — нет в игре)

Шаг 5: (Маньяк пропускается — нет в игре)

Шаг 6: Озвучка "Комиссар просыпается."
        Денис → NightCommissionerScreen
        Остальные → NightWaitScreen "Комиссар ведёт расследование..."
        
        Денис проверяет Максима → "Мафия!"
        Денис нажимает "Понятно"
        
        Озвучка "Комиссар засыпает."

Шаг 7: Озвучка "Врач просыпается."
        Юля → NightDoctorScreen
        Остальные → NightWaitScreen "Врач спасает жизни..."
        
        Юля выбирает вылечить Дениса
        
        Озвучка "Врач засыпает."

Шаг 8: Разрешение ночных действий:
        - Мафия убивает Дениса
        - Врач лечит Дениса
        → Денис СПАСЁН!
        
        Озвучка "Город просыпается. Наступает день."
```

## Сценарий 4: Дневная фаза

### Пример: День 1 (после ночи из сценария 3)

```
Шаг 1: DayResultsScreen
        Все видят: "Этой ночью в городе было тихо. Никто не погиб."
        (Денис выжил благодаря Врачу)
        Хост нажимает "Начать обсуждение"

Шаг 2: DiscussionScreen
        Таймер: 2:00
        Игроки обсуждают вживую
        Хост нажимает "Начать голосование" (в любой момент)

Шаг 3: VotingScreen
        Каждый выбирает за кого голосовать:
        
        Артём → голосует за Максима
        Марина → голосует за Артёма
        Денис → голосует за Максима (знает что он мафия!)
        Алина → голосует за Максима
        Костя → голосует за Артёма
        Юля → голосует за Максима
        Максим → голосует за Дениса
        Катя → пропускает
        
        Результат (открытое голосование):
        Максим: 4 голоса (Артём, Денис, Алина, Юля)
        Артём: 2 голоса (Марина, Костя)
        Денис: 1 голос (Максим)
        
        → Город изгоняет Максима!

Шаг 4: VoteResultOverlay
        "Город изгоняет Максима!"
        Роль: Мафия (если настройка включена)
        Хост нажимает "Продолжить"

Шаг 5: LastWordScreen
        Максим произносит прощальное слово (таймер 30 сек)
        Хост нажимает "Продолжить"
        
        Максим → DeadScreen
        
        Проверка победы:
        Мафия живы: Марина, Костя (2)
        Остальные живы: Артём, Денис, Алина, Юля, Катя (5)
        2 < 5 → Игра продолжается

Шаг 6: → Следующая ночь (Ночь 2)
```

## Сценарий 5: Победа мафии

### Предусловия
- Ночь 3
- Живы: Марина (Мафия), Костя (Дон), Артём (Мирный), Алина (Мирный), Катя (Мирный)
- Мафия: 2, Остальные: 3

```
Ночь 3:
  Мафия убивает Катю
  Врач мёртв → никто не лечит
  
День 3:
  "Этой ночью была убита Катя."
  Живы: Марина, Костя, Артём, Алина
  Мафия: 2, Остальные: 2
  
  → 2 >= 2 → МАФИЯ ПОБЕЖДАЕТ!
  → GameOverScreen: "Мафия победила!"
  
  Раскрытие ролей:
  Марина — Мафия
  Костя — Дон мафии
  Артём — Мирный житель
  Денис — Комиссар (убит ночью 2)
  Алина — Мирный житель
  Юля — Врач (изгнана днём 2)
  Максим — Мафия (изгнан днём 1)
  Катя — Мирный житель (убита ночью 3)
```

## Сценарий 7: Игра с Маньяком

```
Роли: 2 Мафии, 1 Дон, 1 Комиссар, 1 Врач, 1 Маньяк, 4 Мирных (10 игроков)

Ночь 1:
  Мафия убивает Игрока_A

  Мафия убивает Игрока_C

  Комиссар проверяет Адвоката → "Мирный" (самозащита)

## Сценарий 9: Таймер истёк (без действия)

```
Ночь — фаза Врача:
  Таймер: 30 сек
  Врач не успевает сделать выбор
  → Таймер истёк
  → Врач НЕ лечит никого этой ночью
  → Игра продолжается

Ночь — фаза Мафии:
  Мафия не определилась с жертвой до конца таймера
  → Случайная жертва из живых не-мафиози
  → Игра продолжается

Голосование:
  Игрок не проголосовал до конца таймера
  → Считается как "воздержался"
```

## Сценарий 10: Отключение игрока

```
Во время игры Марина теряет соединение:

1. Хост получает событие disconnect
2. На экране хоста: модальное окно "{Марина} отключилась. Ожидание 30 сек..."
3. Игра ставится на паузу (таймеры остановлены)

Вариант А: Марина переподключается за 30 сек
  → Марина открывает приложение
  → Автоматически подключается к комнате (если была вкладка открыта)
  → Или вводит код комнаты + своё имя
  → Хост распознаёт игрока по имени → восстановление сессии
  → Игра продолжается

Вариант Б: Марина не переподключается
  → Хост нажимает "Кикнуть"
  → Марина считается выбывшей (мёртвой? или просто убрана?)
  → Если Марина — мафия, это влияет на баланс
  → Хост решает: продолжить или завершить игру
```

## Сценарий 12: Новая игра после завершения

```
1. GameOverScreen → все видят результаты
2. Хост нажимает "Новая игра"
3. Все игроки возвращаются в LobbyScreen
4. Роли сброшены
5. Хост может изменить настройки
6. Новые игроки могут присоединиться
7. Хост нажимает "Начать" → новая раздача ролей

## Сценарий 13: Первый ход мёртвой роли

```
Ночь 3:
  Комиссар был убит на день 2 → мёртв
  
  Фаза "Комиссар просыпается" ПРОПУСКАЕТСЯ полностью:
  - Нет озвучки "Комиссар просыпается" / "Комиссар засыпает"
  - Нет паузы
  - Сразу переход к следующей фазе (Врач)
  
  На экране ожидания: фаза комиссара не отображается
  → Это быстрее, но может выдать информацию (игроки заметят что фаза быстрее)
  
  РЕШЕНИЕ: Даже для мёртвых ролей выдерживается минимальная пауза (3–5 сек)
  с озвучкой, чтобы живые не могли определить, жива ли роль
```
